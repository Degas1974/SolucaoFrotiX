# 2026.01.21 - Perfil IIS Express Limpo

## Resumo

- Objetivo: criar um perfil de depuração mais silencioso para IIS Express.
- Decisões: desativar Browser Tools e Hot Reload via variáveis de ambiente no profile dedicado.

## Arquivos alterados

- `Properties/launchSettings.json`

## Observações

- Use o profile **IIS Express (Limpo)** para reduzir warnings de BrowserLink/BrowserRefresh.
