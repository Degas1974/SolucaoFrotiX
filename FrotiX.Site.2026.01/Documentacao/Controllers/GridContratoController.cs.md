## ✅ Visão Geral

Controller API que retorna itens de contrato via classe auxiliar `ItensVeiculo`.

---

## 🔧 Endpoints Principais

- `DataSource`: retorna lista formatada para grid.

---

```csharp
[Route("DataSource")]
```

- Tratamento de erro retorna `View()` conforme padrão do arquivo.
