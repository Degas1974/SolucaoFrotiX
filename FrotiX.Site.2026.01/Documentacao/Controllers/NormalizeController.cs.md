# NormalizeController.cs — Normalização de texto

> **Arquivo:** `Controllers/NormalizeController.cs`  
> **Papel:** endpoint para normalizar texto via `NormalizationService`.

---

## ✅ Visão Geral

Controller API que recebe texto e retorna versão normalizada (acentos, padrões), com validação simples.

---

## 🔧 Endpoint Principal

- `POST /api/Normalize`: normaliza texto recebido.

---

## 🧩 Snippet Comentado

```csharp
[HttpPost]
public async Task<ActionResult<string>> Post([FromBody] NormalizeRequest body)
{
    if (body is null || string.IsNullOrWhiteSpace(body.Text))
        return BadRequest("Text is required.");

    var result = await _normalizer.NormalizeAsync(body.Text);
    return Ok(result);
}
```

---

## ✅ Observações Técnicas

- Retorna erro 500 com mensagem genérica quando falha.
- Usa `Alerta.TratamentoErroComLinha` no catch.
