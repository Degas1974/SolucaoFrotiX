# PARTE 1: VISÃO GERAL

Serviço de processamento e cache de fotos de motoristas com redimensionamento automático.

## Características
- Cache de 1 hora
- Redimensionamento para 60x60px
- Formato: data URI base64
- Requer Windows (System.Drawing)
