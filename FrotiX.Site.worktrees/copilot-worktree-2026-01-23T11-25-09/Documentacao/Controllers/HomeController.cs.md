# HomeController.cs — Grid demo / datasource

> **Arquivo:** `Controllers/HomeController.cs`  
> **Papel:** endpoints de demo para grids e datasource.

---

## ✅ Visão Geral

Controller API que expõe endpoints para grid e operações CRUD em memória usando `OrdersDetails`.

---

## 🔧 Endpoints Principais

- `Index`: view padrão.
- `DataSource`: retorna lista de `OrdersDetails`.
- `UrlDatasource`: paginação e contagem.
- `CrudUpdate`: insert/update na lista estática.

---

## 🧩 Snippet Comentado

```csharp
[Route("DataSource")]
[HttpGet]
public IActionResult DataSource()
{
    var order = OrdersDetails.GetAllRecords();
    return Json(order);
}
```

---

## ✅ Observações Técnicas

- Dados são mantidos em lista estática em memória.
- Tratamento de erro retorna `View()` padronizado.
