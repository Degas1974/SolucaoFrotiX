Agenda - Index (Técnico)

Agenda de Viagens · Recorrência e modal de agendamento · Versão 5.4

Visão Geral

View principal da agenda com calendário FullCalendar e modal complexo de agendamento/viagem. Regras de recorrência são controladas por módulos JS específicos.

  • Calendário interativo com edição de eventos.
  • Modal de recorrência com períodos Diário/Semanal/Quinzenal/Mensal/Variada.
  • Integração Syncfusion para DatePickers e DropDowns.
Data Final Recorrência (UI)

Em modo de edição, a Data Final Recorrência pode ser exibida como texto readonly para evitar falhas de inicialização do DatePicker.

CSS: .ftx-final-recorrencia-texto - text-align: left - width: 50% - margin: 0 JS: mostrarTxtFinalRecorrencia() - Alterna DatePicker/texto conforme flags de edição CSS: .ftx-ocultar-recorrencia-date - display: none !important CSS: .ftx-ocultar-recorrencia-texto - display: none !important JS: MutationObserver (ocultação persistente) - Reaplica display none quando Syncfusion reescreve o wrapper
