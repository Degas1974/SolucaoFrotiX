# 2026.01.15 - ItensContrato Status e Encarregados

## Resumo Executivo

Foram habilitados os badges de status clicáveis na página `Contrato/ItensContrato` para Encarregados, Operadores, Motoristas e Lavadores, seguindo o mesmo comportamento já usado em veículos. Na página `Encarregado/Index`, os badges de status passaram a usar as classes padrão do módulo de contratos, e o comentário de documentação perdido no frontend foi removido. Também foram atualizadas as documentações técnicas e criada a versão em HTML A4.

## Arquivos Alterados

1. `wwwroot/js/cadastros/itenscontrato.js`:
   - Adicionados handlers de clique para atualização de status de encarregados, operadores, motoristas e lavadores.
   - Renderização dos badges atualizada para usar links clicáveis e classes padrão.

2. `wwwroot/js/cadastros/encarregado.js`:
   - Alinhadas classes dos badges de status com o padrão de `ItensContrato`.
   - Removido bloco de comentário de documentação perdido no arquivo.

3. `Documentacao/Pages/Encarregado - Index.md` e `Documentacao/Pages/Contrato - ItensContrato.md`:
   - Atualizadas versões, datas e logs de modificações.

4. `Documentacao/Pages/Encarregado - IndexA401.html`:
   - Criada documentação visual em formato A4 com resumo das alterações.

5. `Documentacao/Pages/Contrato - ItensContratoA401.html`:
   - Criada documentação visual em formato A4 com foco no status clicável.

## Decisões Técnicas

- **Badge de status como link**: adotado o padrão já utilizado em veículos (`btn btn-verde`/`fundo-cinza`), garantindo consistência visual e ação imediata.
- **Handlers separados por tipo**: mantidos endpoints específicos para cada entidade (Encarregado, Operador, Motorista, Lavador), reduzindo risco de regressão.
