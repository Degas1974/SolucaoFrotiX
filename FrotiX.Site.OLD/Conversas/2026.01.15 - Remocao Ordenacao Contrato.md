# 2026.01.15 - Remoção de Ordenação Visual no Header de Contratos

## Resumo Executivo
Foi solicitada a remoção das setas de ordenação do cabeçalho da tabela de contratos na página `Contrato/Index`. A alteração foi realizada desabilitando a funcionalidade de ordenação do DataTable, uma vez que os dados já são fornecidos ordenados pelo backend.

## Arquivos Alterados
1.  `wwwroot/js/cadastros/contrato.js`:
    -   Alterada configuração do DataTable: `ordering: false` substituindo `order: [[0, "desc"]]`.
    -   Isso remove os indicadores visuais (setas) e a interatividade de ordenação.

2.  `Documentacao/Pages/Contrato - Index.md`:
    -   Atualizada a versão do documento para 2.1.
    -   Atualizada a data de última atualização.
    -   Atualizado o snippet de código JS na documentação.
    -   Adicionado registro no Log de Modificações.

## Decisões Técnicas
-   **Desativação da Ordenação (`ordering: false`)**: Optou-se por desativar globalmente a ordenação na tabela ao invés de apenas ocultar setas via CSS, garantindo que o comportamento corresponda ao visual (sem ordenação interativa) e confiando na ordenação padrão do servidor (`AnoContrato descending`).
