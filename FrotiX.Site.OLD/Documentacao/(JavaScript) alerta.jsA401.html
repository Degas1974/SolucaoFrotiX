alerta.js

Camada JS para exibir alertas SweetAlert/Toasts • Última atualização: 08/01/2026 • Versão 2.0

Visão geral

Abstrai SweetAlert/Toasts no frontend, alinhado aos helpers Alerta/AlertaBackend. Facilita chamadas padronizadas de sucesso/erro/warn/info.

APIs expostas
Alerta.sucesso("Registro salvo");
Alerta.erro("Falha ao salvar");
Alerta.info("Nada a processar");
Alerta.warn("Reveja os campos");
Cuidados
  • Manter textos curtos e consistentes.
  • Sincronizar com sweetalert_interop.js para interop Blazor/JS.
