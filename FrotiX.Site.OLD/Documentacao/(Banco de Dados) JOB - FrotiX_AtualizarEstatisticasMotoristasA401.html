Job - Atualizar Estatísticas de Motoristas

Recalcula dashboards de motoristas (mês atual e anterior) a cada hora

Execução principal
DECLARE @Ano INT = YEAR(GETDATE());
DECLARE @Mes INT = MONTH(GETDATE());
EXEC sp_RecalcularEstatisticasMotoristas @Ano, @Mes;
-- Recalcula mês anterior
SET @Mes = @Mes - 1; IF @Mes = 0 BEGIN SET @Mes = 12; SET @Ano = @Ano - 1; END;
EXEC sp_RecalcularEstatisticasMotoristas @Ano, @Mes;
  • Banco: FrotiX
  • Tentativas: 0 (sem retry); em falha o job termina.
  • Cobre mês atual e anterior (mitiga lançamentos retroativos).
Agendamento (SQL Agent)
  • Diário (freq_type=4), intradiário a cada 1 hora (freq_subday_type=8, freq_subday_interval=1).
  • Janela: 00:00:00 a 23:59:59.
  • Vigência: início 2026-01-01, sem fim.
  • Servidor: (local).
Sem retries configurados

Considerar retry para resiliência e evitar sobreposição com outros jobs horários.
