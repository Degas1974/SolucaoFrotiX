Registro de Cupons (Upsert)

Upload e indexação de lotes de cupons de abastecimento · Versão 1.0

Visão Geral

Interface dedicada para a criação e edição de registros de cupons. Combina upload de arquivo PDF, visualização imediata para conferência e editor de texto rico para notas.

  • Upload PDF Kendo UI Upload com pré-visualização
  • Editor Texto Syncfusion RTE para observações
Fluxo de Upload

O arquivo é enviado para um Handler específico e visualizado logo em seguida.

// Sucesso do Upload function onSuccess(e) { var files = e.files; $.each(files, function () { // Inicializa Viewer $("#pdfViewer").kendoPDFViewer({ pdfjsProcessing: { file: "/DadosEditaveis/Cupons/" + this.name } }); }); }
Handlers Razor Pages
  • POST ?handler=SavePDF - Salva o arquivo no disco.
  • POST ?handler=RemovePDF - Remove arquivo (limpeza).
Edição Rica

O campo de observações permite formatação HTML completa para detalhar o conteúdo do lote de cupons.
