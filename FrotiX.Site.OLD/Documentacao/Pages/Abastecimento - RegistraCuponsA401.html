Registro de Cupons

Gestão documental de abastecimentos digitalizados · Listagem · Versão 1.0

Visão Geral

Ponto de entrada para a gestão de documentos de abastecimento. Permite localizar lotes de cupons digitalizados, visualizar o PDF associado e realizar a manutenção (edição/exclusão).

  • PDF Visualizador Kendo UI integrado
  • DataTable Listagem rápida com paginação
Filtros e Ações

Interface simplificada focada na data do registro.

  • Filtro Data: Selecione o dia para ver os lotes registrados.
  • Visualizar: Clique no registro para abrir o modal com PDF.
  • Excluir: Remoção física do registro e arquivo (com confirmação).
API Endpoints
  • GET /ListaRegistroCupons - Todos os registros.
  • GET /PegaRegistroCuponsData?id=DD/MM/YYYY - Filtro por data.
  • GET /DeleteRegistro?IDapi={guid} - Exclusão.
  • GET /PegaRegistroCupons?IDapi={guid} - Detalhes para modal.
Visualizador PDF

Utiliza Kendo UI PDF Viewer apontando para /DadosEditaveis/Cupons/.

$("#pdfViewer").kendopdfViewer({ pdfjsProcessing: { file: "/DadosEditaveis/Cupons/" + data.registroPDF } });
