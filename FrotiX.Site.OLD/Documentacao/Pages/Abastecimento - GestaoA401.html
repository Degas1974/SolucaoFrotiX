Gestão de Abastecimento

Controle central de abastecimentos, edição de KM e exportação · Versão 1.0

Visão Geral

Gerenciamento completo dos abastecimentos da frota. Permite listar, filtrar com precisão (por Unidade, Motorista, Veículo), editar quilometragens erradas e exportar relatórios.

  • DataTable Listagem robusta com filtros e paginação
  • Syncfusion ComboBoxes de filtro avançados
  • Edição Modal para correção de KM Rodado
  • Export Excel (.xlsx) e PDF Paisagem
Arquitetura e API

Padrão MVC com API REST para dados dinâmicos. Frontend consome JSON.

EndpointDescrição
GET /api/abastecimentoRetorna todos os registros
GET /AbastecimentoDataFiltra por Data (DD/MM/YYYY)
GET /AbastecimentoVeiculosFiltra por ID Veículo
POST /EditaKmAtualiza KM (Body JSON)

Troubleshooting
  • DataTable não carrega: Verificar console para erro 500 ou JSON malformado.
  • Modal não limpa: O evento hide.bs.modal deve resetar o input e o estado do botão.
  • Spinner travado: Ocorreu erro no AJAX sem catch. Verificar Alerta.TratamentoErroComLinha.
