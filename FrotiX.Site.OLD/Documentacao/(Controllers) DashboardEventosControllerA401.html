DashboardEventosController

APIs de dashboard de eventos · Última atualização: 08/01/2026 · Versão: 2.0

Visão geral

Retorna dados para o dashboard de eventos (contagens, séries e agrupamentos). Consumido por páginas de dashboards e widgets.

Endpoints típicos
  • GET /api/DashboardEventos — métricas agregadas (por período).
  • GET /api/DashboardEventos/Series — séries temporais.
  • GET /api/DashboardEventos/Resumo — resumo por categoria/status.

Nomes podem variar; foco é entregar dados agregados para gráficos.

Snippet (consulta agrupada)
// Conceito: agrupar eventos por status
var dados = await _unitOfWork.Evento.GetAllAsync();
var grouped = dados
.GroupBy(e => e.Status)
.Select(g => new { Status = g.Key, Total = g.Count() })
.ToList();
return Ok(new { data = grouped });
Interconexões
  • Repos: `_unitOfWork.Evento` e correlatos.
  • Pages: dashboards de eventos (gráficos/series).
  • Dependências visuais: widgets/JS que consomem JSON para gráficos.
