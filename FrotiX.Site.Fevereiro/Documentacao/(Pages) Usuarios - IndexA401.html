Usuarios - Index

Listagem e Gestao de Usuarios do Sistema FrotiX

Visao Geral

A pagina Usuarios/Index e o painel central de gestao de usuarios do FrotiX. Ela exibe todos os usuarios cadastrados em uma DataTable interativa, permitindo visualizacao, edicao, gerenciamento de recursos de acesso e exclusao de usuarios.

Listagem Edicao Recursos Exclusao
Destaque: A pagina utiliza DataTables para exibicao interativa com ordenacao, filtro e paginacao. Os avatares dos usuarios sao exibidos junto ao nome, e o sistema de badges permite alterar status com um clique.
Arquitetura

Estrutura de arquivos envolvidos nesta funcionalidade:

Modais da Pagina

Modal Controle de Acesso

Permite gerenciar os recursos (paginas/funcionalidades) que o usuario pode acessar. Exibe uma DataTable com todos os recursos do sistema e badges clicaveis para conceder/remover acesso.

  • Header: Gradiente azul petroleo (#3D5771 a #2d4559)
  • Titulo: "Gestao de Recursos do Usuario"
  • Botao Fechar: btn-vinho com icone fa-circle-xmark

Modal Foto do Usuario

Exibe a foto do usuario em tamanho ampliado, ou um placeholder caso nao tenha foto cadastrada.

  • Header: Gradiente verde escuro (#2F4F4F a #1a3030)
  • Titulo: "Foto do Usuario"
  • Botao Fechar: btn-vinho com icone fa-circle-xmark
Endpoints API Utilizados
Metodo Endpoint Descricao
GET /api/Usuario/GetAll Lista todos os usuarios com foto em Base64
GET /api/Usuario/PegaRecursosUsuario Busca recursos de um usuario especifico
GET /api/Usuario/UpdateStatusUsuario Alterna status Ativo/Inativo
GET /api/Usuario/UpdateCargaPatrimonial Alterna Detentor de Carga Patrimonial
GET /api/Usuario/UpdateStatusAcesso Alterna acesso a um recurso
POST /api/Usuario/Delete Exclui um usuario
