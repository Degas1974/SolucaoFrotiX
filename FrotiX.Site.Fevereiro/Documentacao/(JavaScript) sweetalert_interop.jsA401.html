sweetalert_interop.js

Interop SweetAlert (JS/Blazor) • Última atualização: 08/01/2026 • Versão 2.0

Visão geral

Fornece funções JS para serem chamadas via interop (ex.: Blazor) usando SweetAlert. Complementa alerta.js.

APIs
SweetAlertInterop.sucesso("ok");
SweetAlertInterop.erro("falhou");
SweetAlertInterop.confirm("Confirma?", callback);
Cuidados
  • Sincronizar mensagens com backend.
  • Usar callbacks seguros ao confirmar.
