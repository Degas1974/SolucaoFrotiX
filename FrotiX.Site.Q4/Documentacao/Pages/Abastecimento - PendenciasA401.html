Pendências de Abastecimento

Gestão de erros de importação com correções inteligentes · Versão 2.0

Visão Geral

Área dedicada ao tratamento de registros que falharam na validação automática durante a importação. Oferece ferramentas poderosas para corrigir dados sem necessidade de reimportar a planilha.

  • Correção KM Sugere automaticamente KM Anterior/Atual
  • Validação Checa Motorista, Veículo e Combustível
  • Workflow Edição -> Revalidação -> Importação
Fluxo de Resolução
1. Identificação: Sistema categoriza erro (KM, Veículo, etc) e gera badge visual.
2. Análise: Usuário visualiza o erro. Se for KM, sistema oferece sugestão.
3. Ação: Usuário edita manualmente ou aplica sugestão automática.
4. Importação: Registro corrigido é movido para tabela de Abastecimentos (Status 1).
Sugestões Inteligentes

O algoritmo analisa a média de consumo do veículo para propor correções:

// Exemplo de Lógica if (KmRodado < 0) { sugestao = KmAtual - (Litros * MediaConsumo); // "Baseado na média de 11 km/l..." }
API Endpoints
  • GET /ListarPendencias - Retorna erros ativos (Status=0)
  • POST /AplicarSugestao - Corrige automaticamente
  • POST /ResolverPendencia - Valida final e importa
