DashboardVeiculosController

APIs de dashboard de veículos (snapshots e rankings) · Última atualização: 08/01/2026 · Versão: 2.0

Visão geral

Fornece dados agregados de veículos: snapshots, rankings, uso, consumo, status. Alimenta dashboards e widgets.

Endpoints típicos
  • GET /api/DashboardVeiculos — KPIs gerais (ativos, consumo, uso).
  • GET /api/DashboardVeiculos/Series — séries por período (uso/consumo).
  • GET /api/DashboardVeiculos/Rankings — rankings por consumo/uso/km.
Snippet (conceito)
var estat = await _unitOfWork.EstatisticaVeiculoUsoMensal
.Where(e => e.Ano == ano && e.Mes == mes)
.ToListAsync();
var topUso = estat.OrderByDescending(x => x.KmRodado).Take(10);
return Ok(new { ranking = topUso });
Interconexões
  • Views/Tabelas: EstatisticaVeiculoUsoMensal, EstatisticaVeiculoGeral, rankings por categoria/modelo/combustível.
  • Jobs/SPs: sp_AtualizarEstatisticasVeiculosMesAtual, sp_RecalcularEstatisticasVeiculo*, sp_RecalcularRankingsVeiculoAnual.
  • Pages: dashboards de veículos.
